dropwizard-redis
================

TBF

### Usage

```xml
	<repository>
		<id>git-47billion</id>
		<name>47billion's Git based repo</name>
		<url>https://github.com/47billion/maven-repo/raw/master</url>
	</repository>
```

```xml
    <dependencies>
        <dependency>
            <groupId>com.lab47.dropwizard</groupId>
            <artifactId>redis</artifactId>
            <version>0.7.1</version>
        </dependency>
    </dependencies>
```
